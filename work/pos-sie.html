POS-SIE

About Project

Aplikasi POS-SIE dibuat untuk mempermudah pengusaha UMKM, aplikasi ini menggabungkan Point Of Sale (POS) untuk membantu seorang kasir mengelola proses transaksi, dan Sistem Informasi Eksekutif (SIE) untuk membantu seorang eksekutif melakukan melakukan analisis terhadap data untuk perkembangan kafe
