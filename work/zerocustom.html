About Project

Aplikasi Rekap Surat BTN dibuat untuk mempermudah user / developer dalam mengajukan dokumen untuk bekerja sama dengan pihak BTN. Dokumen tersebut akan di cek terlebih dahulu oleh admin, jika admin menyetujui maka status dokumen user akan menjadi "Approved" dan jika tidak maka statusnya akan menjadi "Rejected".

Aplikasi ini juga dapat merekap data-data user / developer yang mengajukan dokumen, maka dari itu jika user / developer ingin melakukan kerja sama dengan pihak BTN lagi, maka user / developer tidak perlu mengajukan dokumen lagi dikarenakan data-data sebelumnya sudah tercatat data database.
